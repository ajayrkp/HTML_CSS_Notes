        CSS Selectors

      • Universal selectors -- Applies to all element - e.g. * {}
      • Type Selector -- Matches Element names - e.g. h1, h2, {}
      • Class Selector -- specify class - e.g. .note {}
      • ID Selector -- element whose ID attribute has a value - e.g. #idName {}
      • Child Selector -- Direct child of another - e.g. li>a {}
      • Descendant Selector -- Descendent of another - e.g. p a {}
      • Adjacent Sibling Selector -- Next sibling of another - e.g. h1+p {}
      • General Sibling Selector - sibling of another, although doesn't have to be directly preceding element - e.g. h1~p {}

        How CSS Rules Cascade

      • Last Rule -- If identical, latter of two will take precedence
      • Specificity -- If one selector is more specific than other, specific one will take precedence
      • Important -- can add !important after any property value to indicate to take higher precedence
      • Example: cascade.html

        Inheritance

      • font-family properties applies to child elements
      • background-color or border properties are NOT inherited by child elements
      • Although, we can force properties to be inherited using 'inherit' for the values of properties
      • Example: inheritance.html

    • Chapter 12 - Text

        • General Font Knowledge

      • Serif: Extra details at the end of main strokes -- used for long passages
          • Example: Georgia, Times, Times New Roman
      • Sans-Serif: Straight ends to letters and much cleaner design
          • Example: Arial, Verdana, Helvetica
      • Monospace: fixed-width font
          • Example: Courier, Courier New
      • Cursive: either have joining strokes or other cursive characteristics, such as handwriting styles
          • Example: Comic Sans MS, Monotype Corsiva
      • Fantasy: Decorative fonts and often used for titles
          • Impact, Haettenshweiler


        Example of using different font properties

      • 'font-family': can specify multiple fonts. Example:font-family.html
      • 'font-size': pt, px, %, em -- Example:font-size.html
          • %: Relative to default font-size on the browser
          em: Relative to size of parent element
          72px = 1 inch (mostly)
      • @font-face: Use for external fonts. Example: font-face.html
      • 'font-weight:normal/bold;' - To make text normal/bold
      • 'font-style: normal/italic/oblique' - change font-style
      • 'text-transform:uppercase/lowercase/capitalize' - Transform Text
      • 'text-decoration:none/underline/overline/line-through/blink' - Specify text decoration
      • 'line-height:1.4em;' - Vertical line height (called Leading)
      • 'letter-spacing':0.2em; - Spacing between letters (called Kerning)
      • 'word-spacing':1em; - Spacing between words (called Kerning)
      • 'text-align:left/right/center/justify' - Self explained
      • 'vertical-align:baseline/sub/super/top/text-top/midle/bottom/text-bottom' - generally used with 'img','em','strong'
      • 'text-indent':20px; - Indent first line of text
      • 'text-shadow': 1px 1px 3px #666666;} -- Example:text-shadow.html


        Pseudo-elements

      • ':first-letter' - Property of first letter
      • ':first-line' - Property of first line
      • ':link' - set styles for link
      • ':visited' - set styles for visited links
      • ':hover' - set styles for hover an element
      • ':active' - set styles for active element
      • ':focus' - set styles when element has focus


        Attribute Selectors

      • 'Existence' - [] matches a specific attribute. Example: p[class]
      • 'Equality' - [=] matches specific attribute with specific value. Example: p[class="val"]
      • 'Space' - [~=] in a space-separated list of words. Example:p[class~="val"]
      • 'Prefix' - [^=] attribute whose value begins with specific string. Example: p[attr^"d"]
      • 'Substring' - [*=] contains specific substring. Example: p[attr*"do"]
      • 'Suffix' - [$=] value ends with specific string. Example: p[attr$"g"]

    • Chapter 13 - Boxes

        • Basic Properties - Width, Height, Border, Padding and Margin

      • 'width, height' - Control width and height of this box
      • 'min-width, max-width, min-height, max-height' - Limiting dimensions
      • 'overflow :hidden/scroll' - tells browser what to do when content larger than box itself
      • 'border-width' - Border of a box... can be defined in px, thin/medium/thick or top,right,bottom,left values for example
      • 'border-style :solid/dotted/dashed/groove/ridge/inset/outset/hidden/none' - Style of border
          • Example: border-style.html
      • 'border-color' - Color of border -- can be controlled for top,right,bottom,left
      • 'border: width style color' - border can be defined using this shorthand tag
      • 'padding: 10px' - Space between border and any content inside box
          • - can be defined for top,,right,bottom,left
      • 'margin: 10px' - sit outside the edge of the border
          • - can be defined for top,,right,bottom,left


        Other Properties -

      • 'margin: xx auto xx auto' - Centers the box [left and right margin set to auto]
      • 'display: inline/block/inline-block/none' - turn inline element into block element or vice versa
      • 'visibility: hidden/visible' - hide/show element


        CSS3 Properties -

      • 'border-image: ' - slice image into 9 pieces and applies to box borders
          • Params: URL, Where to slice image, What to do with the straight edges [stretch, repeat, round]
      • 'box-shadow: ' - drop shadow around a box
          • Params: Horizontal Offset, Vertical Offset, Blur distance, spread of shadow, COLOR
      • 'border-radius: ' - Create rounded corners
          • Params: for - top-right, bottom-right, bottom-left, top-left
          - Can be used to create Elliptical shapes using different radius of particular corners

    • Chapter 15 - Layout

        • Position of Elements

      • 'position:static;' - Normal flow [each element sits on top of one another]
      • 'position:relative;' - Relative positioning [in relation to it would have been in normal flow]
          • - Use top/ left/ bottom/ right properties along with
      • 'position:absolute;' - Absolute positioning [In relation to it's containing element]
          • - Use top/ left/ bottom/ right/ width properties along with
      • 'position:fixed;' - Fixed positioning [In relation to the browser window]
          • - Use top/ left/ bottom/ right/ padding/ margin/ background-color/ width properties along with
      • 'z-index;' - For Overlapping elements [Decides which element sits on top]
          • - Param: 'number': higher the number, closer to front
      • 'float;' - Floating elements. [For normal flow]
          • - Param: left/ right/ top/ bottom/ width/ margin/ padding etc...
          - Float can used to place elements side-by-side
      • 'clear' - Clearing floats
          • - Param: left/ right/ both/ none
      • 'Multicolumn layouts with floats-- using div'
          • - Param: width/ float/ margin ... and offcourse use 'div'
          - Similarly 3 column layout can be created using 3 'div'


        Handle different size of the screen

      • 'Liquid layout is better than fixed layout for different size of screens'
          • - For fixed layouts, fixed value of px can be used
          - For relative layouts, percentage used


        Layout Grids

    • Chapter 16- Images

        • Specify the size and alignment of an image

      • 'size' - width, height [just like box properties]
          • - Whenever use consistently sized images across a site, we can use CSS to control the dimensions of the images, instead of putting the dimensions in the HTML.
      • 'align' - Use 'float' property
          • - Use margin to make sure images doesn't touch the edges
      • 'Centering image' - How to center the image
          • - 'display: block;' makes an image as block element
          - use 'text-align: center' or 'margin: auto auto;' to center the image


        Background images

      • 'background-image: url("");' - Background image behing an HTML element
          • - by default image repeats in entire box
      • 'background-repeat' - Background repeats in following fashion
          • - Params: repeat-x (repeat horizontally), repeat-y, no-repeat
      • 'background-attachment' - specifies whether a background image should stay in one position or move as the user scrolls up and down the page.
          • - Params: fixed, scroll
      • 'background-position' - Specifies where in browser window this background image can be placed
          • - Params: left top/ left center/ left bottom/ center top/ center center/ center bottom/ right top/ right center/ right bottom
          - Params: or percent (distance from top-left corner of browser window)
      • 'background' - shorthand


        Other Image properties

      • 'Image rollover and sprites' - Following very good example, where just one image is used for two button and 6 different images
          • - Example: image-rollovers-and-sprites.html
      • 'CSS3: Gradients' - 'background-image: ..linear-gradient()'
          • - Specify two colors for linear gradient
